## 📊 서비스 배경

### 전세사기 현황
- 2024년 기준 전세사기 피해자 약 2만명
- 20~30대 청년층이 전체 피해자의 75% 차지
  - 20대: 약 5천명 (25%)
  - 30대: 약 1만명 (50%)
- 임차권등기명령 신청 58% 증가
  - 2023년 1-4월: 11,339건
  - 2024년 1-4월: 17,917건

## 🔧 기술 아키텍처
AWS 클라우드 기반의 서버리스 아키텍처 구현
- Frontend: React.js
- Backend: AWS Lambda, API Gateway
- Storage: Amazon S3
- AI/ML: AWS Chatbot, OCR 서비스
